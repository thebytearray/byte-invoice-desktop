# Byte Invoice Website

Marketing website for [Byte Invoice](https://thebytearray.github.io/byte-invoice/). Built with React, Vite, and Chakra UI.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is in `dist/`. The site is deployed to GitHub Pages via the [Deploy Website](../.github/workflows/deploy-website.yml) workflow.
